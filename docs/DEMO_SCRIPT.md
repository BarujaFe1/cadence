# Demo script — Cadence (~4 minutes)

**Audience:** recruiter / hiring manager exploring a lab side project  
**URL:** https://cadence-ecru-three.vercel.app (or local `pnpm dev`)

## Goals

Show calm UX, the session machine, and honest local-first limits — without framing Cadence as a data product.

## Script

1. **Open (20s)** — “Personal weekday language routine: EN → FR → DE → ES.”  
2. **Start / pause (60s)** — Start English, pause, resume (Space). Point at the progress ring.  
3. **Complete & next (45s)** — Complete (C), next block (N). Mention manual complete as check-off.  
4. **Focus (20s)** — Toggle Foco (F).  
5. **History (60s)** — Streak = weekdays with ≥1 completed entry; weekends don’t break it.  
6. **Settings (30s)** — Change minutes; optional demo seed.  
7. **Close (20s)** — “Local-first on purpose. Lab/side project, not my data thesis.”

## Do not say

- enterprise / premium / production-grade platform  
- analytics engineering / ML / “data pipeline”  
- that sync or accounts exist
